# CTRL-TUI
CTRL-TUI is a TUI (Terminal User Interface) client write form scratch in C++

## INSTALLATION
### REQUIREMENT
- GLIBC
- Make
- A linux system

### STEP
1. Clone the project and go in the project folder
```sh
git clone https://github.com/Gun8hoot/CTRL-TUI.git && cd CTRL-TUI
```
2. Compile the Project
```sh
make
```
